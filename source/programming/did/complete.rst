Deliverable Completion
==================================
The *Resource Allocation and Tracking Flow for Study Support* section of `03 SDSA DID management user manual.pdf <https://pfizer.sharepoint.com/:b:/r/sites/SPAaccelerator/Repository/Pfizer_vendors/03%20SDSA%20DID%20management%20user%20manual.pdf?csf=1&web=1&e=5pWRbJ>`__ describes 
the actions to perform when the deliverable is complete.

#. Update the DID with the actual delivery date. DID status automatically updates to 'Complete'.
#. SDSL updates the quality assessment info. DID sends auto reminder to SDSL to complete this step.
#. Update DID with TMF status.
