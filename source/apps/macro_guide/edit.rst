Macro Library Administration
===========================================
Members of SPI can perform manually add, update, and delete macros and parameters withing macros from the macro library home page. 


Manually Add a Macro
---------------------------
To create a new macro manually click |manual| to create an blank entry in the data grid. Complete the macro level attributes for the macro and click the row-level |rowsave| button 
to add the macro the database.

The grid will autofilter on the new macro to allow parameters to be specified. See `add parameters <edit.html#add-or-update-macro-parameters>`__ for details on managing parameters.

When finished, click |filteroff| to return the unfiltered library.

Update an Existing Macro
----------------------------
Update an existing macro by clicking |rowedit| to enable editing mode. Update the macro as desired and click |rowsave| or |rowcancel| to save or abandon changes. 

Add or Update Macro Parameters
-------------------------------------
To add a parameter, click |expand| to bring up the nested grid, and click |addparm| to
create an empty parameter. Once complete, click |rowsave| to commit the record to the database. If an anytime you want to cancel, click the |rowcancel| to abandon changes.

Delete a Macro
-----------------------------
Delete a macro by clicking |delete| and accepting the confirmation dialog.
