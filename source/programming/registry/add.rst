Add a Context menu
===========================
#. Save the installer file from the `library <library.html>`__ locally for the application by clicking the download button.
#. Extract the .reg from from the .zip file (c:\\users\\[username]\\downloads by default) onto any local folder, not the file share or OneDrive. 
#. Double-click the extracted .reg file and select Yes at the prompts. 
#. Observe success dialog upon completion
   |success|
